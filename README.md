# MDC_cut: An ARPES data analysis tool

The application is designed dedicated to process the data generated by **Spectrium**,
which is a software product of **PREVAC sp. z o.o.**.
It is a useful GUI to process the data acquired by PREVAC EA15 hemispherical electron energy analyzer without any difiiculties.
Please check the file **MDC_cut_UserManual.pdf** for more detailed information.
